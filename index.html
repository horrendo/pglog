    pglog

    A lot of useful information can be obtained from Postgres csv-formatted logs. This shard aims to make that process easier.

    Installation

    1. Add the dependency to your shard.yml:

    `yaml dependencies:

     pglog:
   github: horrendo/pglog

    `

    1. Run shards install

    Usage

    require "pglog"

    You can access the documentation here
